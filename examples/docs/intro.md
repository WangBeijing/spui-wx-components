## 1.使用教程

### 下载`spui-wx-components`
```
npm install spui-wx-components
```
### 项目中使用`spui-wx-components`
```html
import Vue from 'vue'
import spui from 'spui-wx-components'
Vue.use(spui)
import 'spui-wx-components/lib/spui-css/index.css'
```

## 2.npm script命令
```html
npm install
# 启动项目
npm run dev 
# 打包js
npm run build:spui
# 打包css
npm run build:spui-css
```
